Журнал прогресса

Создание концепт-документа (03.02.25 - 26.05.2025)

Был создан концепт-документ, включающий в себя описание всего проекта.

Рефакторинг кода (21.04.2025 - 05.05.2025)

Проведена оптимизация и упрощение кодовой базы, устранены дублирующиеся функции, улучшена читаемость и поддержка кода.

Улучшение UI (07.04.2025 - 21.04.2025)

Доработан пользовательский интерфейс: улучшена навигация, добавлены анимации, исправлены визуальные недочеты для лучшего взаимодействия с игроком.

Расширение текстовых сценариев (24.03.2025 - 07.04.2025)

Дописаны новые диалоговые ветки, добавлены альтернативные реплики и уточнены сюжетные моменты для большей глубины повествования.

Привязка UI к коду (10.03.2025 - 24.03.2025)

Интерфейсные элементы (кнопки, меню, HUD) интегрированы с игровой логикой, обеспечена корректная реакция на действия игрока.

Создание альфа-версии диалоговой системы игры (28.02.2025 - 10.03.2025)

Реализован базовый функционал диалогов: вывод реплик, простые ветвления и сохранение выбора игрока для дальнейшего развития сюжета.
